# ClaudeMD Automation Framework

## Overview
Page Object Model automation framework for ClaudeMD application using Playwright and JavaScript.

## Modules
- Login
- Appointment Board
- PM Board
- Billing

## Test Cases
JSON test case definitions are in testcases/smoke/ folder.

## Setup
1. Copy .env.example to .env
2. Update credentials in .env file
3. Run: npm install

IMPORTANT: .env file contains credentials and is in .gitignore - never commit it to git!
